Open...

Menu Options ›› File ››

To open an existing save file, choose File > Open... or press Ctrl+O. If there is a game with unsaved changes currently in progress, you will be asked if you want to save the game before opening a different one. To save, click Yes (if the match has not been previously saved and you Cancel out of the resulting dialog box, no game will be opened and your current unsaved game will be unaffected). To discard any unsaved changes and open a saved game immediately, click No. To stay with your current unsaved game, click Cancel.
