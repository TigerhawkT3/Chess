New

Menu Options ›› File ››

Start a new game by choosing File > New or pressing Ctrl+N. If there is a game with unsaved changes currently in progress, you will be asked if you want to save the game before starting a new one. To save, click Yes (if the match has not been previously saved and you Cancel out of the resulting dialog box, no new game will be started and your current unsaved game will be unaffected). To discard any unsaved changes and start a new game immediately, click No. To stay with your current unsaved game, click Cancel.
