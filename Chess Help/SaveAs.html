Save As...

Menu Options ›› File ››

To create a new save file for a game, even if you had already saved one earlier, choose File > Save As or press Ctrl+Shift+S (that is, Ctrl+capital S). This can be useful for preserving different portions of a match, exploring different forks (see Navigation), or simply making a copy of a game in a new location without copying and pasting it. See Save for more information about save files.
