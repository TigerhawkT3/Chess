System requirements

Getting Started ››

To play Chess, you will need the following:


  • A computer made in this millennium
  • A keyboard
  • A mouse
  • Speakers and sound card for the sound effects


You can run the source code if you have the following additional items:


  • The latest version of Python 3
  • Pygame (Windows) (other OS)
  • The Pillow fork of PIL (Windows) (other OS)


Unix distributions often include Python, but running the source code may require that you install tkinter with the following steps:


  1. If you are using a Unix distribution with a graphical interface, such as Ubuntu, open a terminal by pressing Ctrl+Shift+T.
  2. At the terminal, enter sudo apt-get install python3-tk and press Enter.


If you'd rather not deal with source code, a bundled package may be available for your system. These packages basically contain the Python interpreter and all necessary modules, allowing you to run Chess without installing anything. The following environments are supported:


  • Windows (32-bit)
  • Windows (64-bit)


If you have a Mac or Linux system and would like to help me make a bundled package for your environment, please let me know.
